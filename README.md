# PhotoGrid

A Python utility to create a photo grid collage from a folder of images.

## Features

*   GUI for selecting an image folder and configuring the grid.
*   Generates a random layout which can be re-generated via a "Shuffle" button.
*   Allows specifying the output image dimensions.
*   Justifies the grid on all four sides by adjusting spacing between photos.
*   Configurable minimum and maximum spacing between images.
*   Preserves the aspect ratio of all source images.
*   Attempts to create a dense, visually pleasing layout.
